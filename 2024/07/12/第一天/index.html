一、注意:
1.安装路径中不要出现中文、空格、特殊字符
2.安装选择空间较大的盘，防止运行时出现蓝屏黑屏
3.下载版本和位数根据自己电脑去选择（本机：win10pro＋64）
4.下载时最好使用火狐下载，关闭软杀
5.绿色版本地址单独列出
二、软件安装：
1.vmware
功能：运行虚拟机
为更好贴近实验版本均采用VM17
许可证：JU090-6039P-08409-8J0QH-2YR7F
官方下载地址：下载 VMware Workstation Pro | CN
内置：kali2023、win10虚拟环境，kali作为攻击机、win10作为日常复现机

2.phpstudy
功能：集成系统环境
此软件为免费软件到官网下载即可，根自身需要去下载内置的数据库、php、服务器

下载地址：小皮面板(phpstudy) - 让天下没有难配的服务器环境！ (xp.cn)
本实验下载地址：https://public.xp.cn/upgrades/phpStudy_64.zip
三、Visual Studio Code
功能：可以运行多个编程语言的软件（本环境主要用于java、python）。

内置插件下载：Chinese、CodeRunner、Codeieum（此插件安装需要访问国外网站，需要挂载vpn，注册账户）配置如下，协议自己选择，服务器ip自己选择，端口选择自己代理服务接受的端口


下载地址：Download Visual Studio Code - Mac, Linux, Windows
